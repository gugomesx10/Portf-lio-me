# Gustavo Gomes — Portfólio Dev Back-end Java

Portfólio pessoal construído com **React + Vite + Tailwind CSS**.

## Tecnologias
- React 18 + Vite 5
- Tailwind CSS 3
- Animações via CSS / IntersectionObserver

## Como rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:5173`

## Assets incluídos
- `public/foto.jpg` — foto de perfil (Hero e Sobre)
- `public/curriculo.pdf` — currículo para download

## Git Flow utilizado
- `main` → produção (tags v1.0.0, v1.0.1)
- `develop` → integração contínua
- `feature/*` → cada seção do portfólio
- `release/1.0.0` → preparação do lançamento
- `hotfix/1.0.1` → correção urgente em produção
